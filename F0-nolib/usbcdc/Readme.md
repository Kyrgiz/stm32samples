Simple code for CAN/USB development board
Simultaneous work of USB CDC (PL2303 emulation) and CAN

Pinout:
PC13 - LED0
PC14 - LED1

PB0..2 - Outputs 1..3

PB8, PB9 - CAN Rx/Tx

PB14 - BRDaddr0
PB15 - BRDaddr1
PA8  - BRDaddr2

PA0 - V12 (external 12V voltage)
PA1 - V5  (5V voltage level)

PA4, PA5 -  Inputs 1,2

PA9, PA10 - USART1 Tx/Rx

PA11. PA12 - USB DM/DP
